## Mission
Build something unexpected that **combines Jupiter APIs in ways they weren't designed for**. The bounty explicitly says: "We've seen enough of those [best app built with Jupiter]. We want you to build something that makes us go 'oh.'" Also: submit a thorough **Developer Experience Report**.

## Track Scope & Focus Areas
- **Creative API Combinations** — chain multiple Jupiter APIs together unexpectedly
- **DX Report** (35% of score!) — honest, specific feedback on developers.jup.ag
- **AI Stack Feedback** (25%) — test Agent Skills, Jupiter CLI, Docs MCP, llms.txt

## SDK / Integration Primitives
Available APIs via `developers.jup.ag` (one API key gets everything):
1. **Swap V2** — `/order` + `/execute` managed routing, or `/build` for raw instructions. Gasless swaps.
2. **Tokens API** — search, metadata, verification status, organic scores, trading metrics
3. **Price API** — USD pricing for all Solana tokens
4. **Lend API** — yield on deposits, borrowing, flashloans
5. **Trigger API** — limit orders: single, OCO (TP/SL), OTOCO **(NEW)**
6. **Recurring API** — time-based DCA
7. **Prediction Markets** — binary markets on real-world events
8. **Perps API** — leveraged perpetuals

AI Stack:
- **Agent Skills** — context files for coding agents
- **Jupiter CLI** — JSON-native, non-interactive terminal for agents
- **Docs MCP** — MCP-based docs querying
- **llms.txt** — LLM-optimized docs index
