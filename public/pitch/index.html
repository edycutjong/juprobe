Key Findings

Real issues discovered from real stress tests against live Jupiter APIs.

🔴 Critical — Silent 429s
The Quote API returns an empty 200 OK body instead of a standard HTTP 429 response when rate limits are breached. This breaks retry-after logic in every major HTTP client library.
🟡 Warning — p95 Latency Spikes
Swap API p50 is stellar at 215ms, but p95 degrades to 854ms during market volatility — causing frontend quote expiration timeouts in optimistic UI implementations.
